# AI-Ready Lead Generation & Instant Response System

This automation ensures businesses never miss a lead by providing instant, real-time engagement via **n8n** and **Webhooks**.

## How it Works
1. **Trigger:** Receives real-time lead data (Name, Email, Phone, Company) via a custom Webhook.
2. **Internal Notification:** Instantly alerts the team via SMTP email with all lead details.
3. **Customer Engagement:** Automatically sends a personalized "Thank You" email to the lead to improve conversion.

## Tech Stack
- **Automation:** n8n
- **Communication:** SMTP / Email nodes
- **Integration:** Webhook (POST method)

## How to Use
1. Download the `lead-gen-workflow.json` file from this repo.
2. Import it into your **n8n** instance.
3. Configure your SMTP credentials in the email nodes.
4. Connect your web form to the Webhook URL.
